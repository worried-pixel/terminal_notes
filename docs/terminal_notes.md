================================================================================
                        TERMINAL NOTES
          The Database That Remembers Everything You Forgot
================================================================================

WHAT IF YOU COULD:
• Search through notes you deleted last year?
• Never lose a file's history when you rename it?
• Jump 5 folder levels deep with one keystroke?
• See every version of every idea you ever had?

That's Terminal Notes. Not another note-taking app. A **temporal database**
that turns Git into a memory for your thoughts.

THE PROBLEM EVERYONE HAS:
=========================
• Git loses history when files move or rename
• Deep folders become navigation nightmares
• Deleted work vanishes forever
• Historical research means switching between tools

OUR SOLUTION:
=============
A terminal interface that makes Git remember what it forgets.

BREAKTHROUGHS (WHY THIS MATTERS):
=================================

1. **GIT THAT REMEMBERS** (Database Research):
   UUIDs in commits = items tracked across time
   Solves Git's fundamental flaw: file ≠ item tracking
   Enables temporal queries academics write papers about

2. **INSTANT NAVIGATION** (HCI Innovation):
   J1/J2/J3 teleportation through infinite folders
   Solves the "deep hierarchy" problem plaguing every file manager
   Terminal UI breakthrough that should be in every IDE

3. **TIME TRAVEL SEARCH** (Information Retrieval):
   Find current, deleted, AND historical items in one query
   What Google Search would be for your personal data
   Research-grade temporal information retrieval

4. **COMPLETE RESURRECTION** (Data Recovery):
   Delete entire project folders → restore with one click
   Not just files - entire hierarchical context returns
   Solves the "archive vs delete" dilemma forever
